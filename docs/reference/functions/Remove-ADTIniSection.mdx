## SYNOPSIS

Opens an INI file and removes the specified section.

## SYNTAX

```powershell
Remove-ADTIniSection [-FilePath] <String> [-Section] <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

Opens an INI file and removes the specified section.

Please note that the INI file provided cannot have a byte order mark (BOM) present as the underlying Win32 API cannot process it correctly.

## EXAMPLES

### EXAMPLE 1

```powershell
Remove-ADTIniSection -FilePath "$env:ProgramFilesX86\IBM\Notes\notes.ini" -Section 'Notes'
```

Removes the 'Notes' section of the 'notes.ini' file.
