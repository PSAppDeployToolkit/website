## SYNOPSIS

Get message for MSI error code

## SYNTAX

```powershell
Get-MsiExitCodeMessage [-MsiExitCode] <Int32> [<CommonParameters>]
```

## DESCRIPTION

Get message for MSI error code by reading it from msimsg.dll

## EXAMPLES

### EXAMPLE 1

```powershell
Get-MsiExitCodeMessage -MsiErrorCode 1618
```

## PARAMETERS
