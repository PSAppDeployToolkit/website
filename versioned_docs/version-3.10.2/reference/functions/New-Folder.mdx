## SYNOPSIS

Create a new folder.

## SYNTAX

```powershell
New-Folder [-Path] <String> [[-ContinueOnError] <Boolean>] [<CommonParameters>]
```

## DESCRIPTION

Create a new folder if it does not exist.

## EXAMPLES

### EXAMPLE 1

```powershell
New-Folder -Path "$envWinDir\System32"
```
