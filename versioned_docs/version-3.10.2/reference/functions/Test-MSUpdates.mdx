## SYNOPSIS

Test whether a Microsoft Windows update is installed.

## SYNTAX

```powershell
Test-MSUpdates [-KBNumber] <String> [[-ContinueOnError] <Boolean>] [<CommonParameters>]
```

## DESCRIPTION

Test whether a Microsoft Windows update is installed.

## EXAMPLES

### EXAMPLE 1

```powershell
Test-MSUpdates -KBNumber 'KB2549864'
```
