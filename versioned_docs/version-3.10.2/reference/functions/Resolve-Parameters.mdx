## SYNOPSIS

Resolve the parameters of a function call to a string.

## SYNTAX

```powershell
Resolve-Parameters [-Parameter] <Object> [<CommonParameters>]
```

## DESCRIPTION

Resolve the parameters of a function call to a string.

## EXAMPLES

### EXAMPLE 1

```powershell
Resolve-Parameters -Parameter $PSBoundParameters | Out-String
```
