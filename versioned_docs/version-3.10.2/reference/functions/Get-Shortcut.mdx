## SYNOPSIS

Get information from a new .lnk or .url type shortcut

## SYNTAX

```powershell
Get-Shortcut [-Path] <String> [-ContinueOnError <Boolean>] [<CommonParameters>]
```

## DESCRIPTION

Get information from a new .lnk or .url type shortcut.
Returns a hashtable.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-Shortcut -Path "$envProgramData\Microsoft\Windows\Start Menu\My Shortcut.lnk"
```
