## SYNOPSIS

Set the history of deferrals in the registry for the current application.

## SYNTAX

```powershell
Set-DeferHistory [[-deferTimesRemaining] <String>] [[-deferDeadline] <String>] [<CommonParameters>]
```

## DESCRIPTION

Set the history of deferrals in the registry for the current application.

## EXAMPLES

### EXAMPLE 1

```powershell
Set-DeferHistory
```
