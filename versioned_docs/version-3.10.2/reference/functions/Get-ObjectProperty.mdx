## SYNOPSIS

Get a property from any object.

## SYNTAX

```powershell
Get-ObjectProperty [-InputObject] <Object> [-PropertyName] <String> [[-ArgumentList] <Object[]>]
 [<CommonParameters>]
```

## DESCRIPTION

Get a property from any object.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-ObjectProperty -InputObject $Record -PropertyName 'StringData' -ArgumentList @(1)
```
