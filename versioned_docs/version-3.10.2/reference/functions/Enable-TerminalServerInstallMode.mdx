## SYNOPSIS

Changes to user install mode for Remote Desktop Session Host/Citrix servers.

## SYNTAX

```powershell
Enable-TerminalServerInstallMode [[-ContinueOnError] <Boolean>] [<CommonParameters>]
```

## DESCRIPTION

Changes to user install mode for Remote Desktop Session Host/Citrix servers.

## EXAMPLES

### EXAMPLE 1

```powershell
Enable-TerminalServerInstallMode
```
